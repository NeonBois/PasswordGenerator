Privacy Policy

1. Password Generation

This Password Generator does NOT generate or access real people's passwords. All passwords are generated randomly using cryptographic-safe methods directly in your browser.

2. Purpose

The generator is intended to help users create strong, unique passwords for personal or testing use. It was created to encourage safe password practices and prevent reuse of weak or common passwords.

3. Data Privacy

No passwords or personal data are collected, stored, or transmitted anywhere. Everything happens locally in your browser.

4. Usage Recommendation

Use the generated passwords responsibly and store them securely. Consider using a password manager for safekeeping.
